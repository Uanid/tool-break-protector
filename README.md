# Tool Break Protector

This mod blocks tool usage, when tool has 2 durability remain.

# Protect Supported Matrix

| Tool     | Action |
|----------|--------|
| Axe      | Mining |
| Pickaxe  | Mining |
| Shovel   | Mining |
| Hoe      | Mining |
| Axe      | Attack |
| Sword    | Attack |
| Bow      | Attack |
| Crossbow | Attack |
| Trident  | Attack |

Protecting `Flint and Steel` and `Shears` are Work In Progress
